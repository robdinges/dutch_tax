================================================================================
DUTCH TAX SYSTEM - INTERACTIVE CALCULATOR
Project Files Summary - February 27, 2026
================================================================================

PROJECT OVERVIEW:
A production-ready Python domain model and interactive form for calculating
Dutch income tax (Box1) and wealth tax (Box3) with realistic 2023-2025 rates.

================================================================================
CORE FILES
================================================================================

1. object_model.py (19 KB)
   - Main domain model with all tax classes
   - Person, Household, Income, Assets, Deductions, Credits
   - Tax calculation methods (Box1, Box3)
   - Full type hints and docstrings
   - Example showing complete household calculation
   
   Key Classes:
   • Person - Individual with income and assets
   • Household - Multiple persons with joint Box3
   • IncomeSource, Asset, Deduction, TaxCredit - Components
   • TaxBracket - Progressive tax bracket definition
   • TaxYearConfig - Complete tax year configuration

2. tax_brackets.py (7.1 KB)
   - Separate tax rate configuration (IMPORTANT for maintenance)
   - Realistic Dutch rates for 2023, 2024, 2025
   - TAX_CONFIGS registry for easy access
   - Functions: get_tax_config(year), get_latest_tax_config()
   
   Available Years:
   • 2023: Tax credit €2,713
   • 2024: Tax credit €2,813
   • 2025: Tax credit €2,917

================================================================================
INTERACTIVE FORMS
================================================================================

3. tax_form.py (12 KB)
   - User-friendly command-line form
   - Guided input for household members
   - Income sources with types
   - Deductions and assets
   - Automatic tax calculation and display
   
   Run: python3 tax_form.py
   
   Features:
   • Input validation and error handling
   • Support for multiple income sources per person
   • Asset categorization (savings, stocks, bonds, real estate, crypto)
   • Box3 allocation strategies (equal/proportional)
   • Formatted output with detailed breakdown

4. form_demo.py (13 KB)
   - Pre-populated demonstrations
   - 3 realistic scenarios:
     1. Simple single-person form
     2. Household with Box3 allocation
     3. Self-employed person with business expenses
   
   Run: python3 form_demo.py
   
   Use For:
   • Understanding form workflow
   • Learning how to structure data
   • Testing complex scenarios

FOR USING THE FORM:
1. python3 tax_form.py               # Launch interactive form
2. Follow prompts to enter data      # Input household info
3. Review calculated results         # See tax calculations

TO ADD A NEW TAX YEAR:
1. Read CONFIGURATION_GUIDE.md       # Get instructions
2. Open tax_brackets.py              # Add new year config
3. Create: create_YYYY_brackets()    # Add bracket function
4. Create: TAX_CONFIG_YYYY instance  # Create config instance
5. Register in TAX_CONFIGS           # Add to registry
6. Done! Use: get_tax_config(YYYY)   # Access new year
